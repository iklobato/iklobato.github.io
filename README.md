## Local Development

```bash
# Clone the repository
git clone https://github.com/iklobato/iklobato.github.io.git
cd iklobato.github.io

# Install dependencies
bundle install

# Start development server
bundle exec jekyll serve

# Visit http://localhost:4000
```
